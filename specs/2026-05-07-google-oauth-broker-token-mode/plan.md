## Group 2 — `GoogleOAuthBroker` class

5. Create `src/brokers/google.py`. Define `@dataclass class GoogleOAuthBroker` with fields: `broker_id: str`, `client_id: str`, `client_secret: str`, `redirect_uri: str`, `default_scopes: list[str]`, plus a private `_token_cache: dict[tuple[str, str], tuple[str, float]] = field(default_factory=dict)` (key `(external_user_id, api_host)`, value `(access_token, expires_at_unix)`).
6. Module-level constants: `_TOKEN_URL = "https://oauth2.googleapis.com/token"` and `_SCOPE_TO_HOST: dict[str, str]` covering the prefixes `https://www.googleapis.com/auth/gmail.*` → `gmail.googleapis.com`, `.../calendar*` → `www.googleapis.com`, `.../drive*` → `www.googleapis.com`, `.../spreadsheets*` → `sheets.googleapis.com`, `.../documents*` → `docs.googleapis.com`, and `openid email profile` → `openidconnect.googleapis.com`. Helper `_scope_to_host(scope: str) -> str | None` does the prefix match; unrecognised scopes return `None` and are dropped during `exchange_code`.
7. Add a small `ExchangeResult` dataclass co-located in `src/brokers/google.py` with fields `provider_user_id: str`, `granted_scopes: list[str]`, `api_hosts: list[str]`.
8. Implement `async def covers(self, api_host: str, external_user_id: str) -> bool` — `SELECT 1 FROM oauth_broker_accounts WHERE broker_id=? AND external_user_id=? AND api_host=? AND refresh_token_enc IS NOT NULL LIMIT 1`. Per-host check (one row per host per the row-shape decision in requirements.md).
9. Implement `async def get_token(self, api_host: str, external_user_id: str) -> str | None` — cache hit (TTL > 60 s) returns cached; else `SELECT refresh_token_enc FROM oauth_broker_accounts WHERE broker_id=? AND external_user_id=? AND api_host=? LIMIT 1`, `vault.decrypt`, POST `_TOKEN_URL` with `grant_type=refresh_token, client_id, client_secret, refresh_token`. On 200: cache the new token, then `UPDATE oauth_broker_accounts SET access_token_enc=?, access_token_expires_at=? WHERE broker_id=? AND external_user_id=?` (single statement updates **all** rows for this Google connection — the access token is shared across hosts in one connection). Return the access token. On 401 / `invalid_grant` return `None`.
10. Implement `async def has_scopes(self, external_user_id: str, required_scopes: list[str]) -> bool` — `SELECT scopes FROM oauth_broker_accounts WHERE broker_id=? AND external_user_id=? LIMIT 1` (any row works — `scopes` is identical across all rows for one connection). Parse JSON, return `set(required_scopes).issubset(set(stored))`. No row → return `False`.
11. Implement `async def proxy_request(self, ...) -> None` — return `None` (Google is token-mode; consumer injects header from `get_token`).
12. Implement `async def exchange_code(self, code: str, external_user_id: str) -> ExchangeResult`. POST `_TOKEN_URL` with `grant_type=authorization_code, code, client_id, client_secret, redirect_uri`. From the 200 response: parse `scope` (space-separated) into a list; resolve unique `api_hosts` via `_scope_to_host` (drop unrecognised scopes); decode `id_token` **defensively** — `split('.')` and validate three segments, pad-fix and `base64.urlsafe_b64decode` the middle segment inside `try/except`, `json.loads` inside `try/except`, validate `sub` is a non-empty string; on any failure raise a sanitised exception that does not echo raw token contents. For each resolved `api_host`, UPSERT one `oauth_broker_accounts` row keyed on the existing `UNIQUE(broker_id, external_user_id, api_host, account_id)` constraint, with `app_slug='google'`, `account_id=<provider_user_id>`, `provider_user_id`, `refresh_token_enc=vault.encrypt(refresh_token)` (always overwrite when a new `refresh_token` is present; **preserve the existing value when the response omits `refresh_token`** — Google omits it on re-grant), `access_token_enc=vault.encrypt(access_token)`, `access_token_expires_at=time.time()+expires_in`, `scopes=json.dumps(granted_scopes)`. Return `ExchangeResult(provider_user_id=sub, granted_scopes=parsed, api_hosts=resolved)`.
13. Implement `@classmethod async def from_db(cls) -> list["GoogleOAuthBroker"]` mirroring `PipedreamOAuthBroker.from_db()` at `src/brokers/pipedream.py:795-829`. `SELECT id, client_id, client_secret_enc, redirect_uri, scopes FROM oauth_brokers WHERE type='google'`, decrypt `client_secret_enc` via `src.vault.decrypt`, parse `scopes` JSON as `default_scopes`, return list of instances.
